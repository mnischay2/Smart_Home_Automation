Institution: Vivekananda Institute of Professional Studies
Academic Year: 2024 – 2025

Description:
------------
Domov is a web-based smart home automation system built using the ESP32 microcontroller and Firebase Realtime Database. It allows users to log in securely via Firebase Authentication and control up to four appliances remotely using a web interface.

Technologies Used:
------------------
- ESP32 (programmed using Arduino IDE)
- Firebase (Realtime Database + Authentication)
- HTML, CSS, JavaScript (Frontend UI)
- Firebase Web SDK
- Hosting via Netlify or Firebase Hosting (optional)

Project Features:
-----------------
- Real-time relay control synced with Firebase
- ESP32 reads Firebase paths to toggle GPIOs
- Clean, responsive UI with toggle buttons
- Logout and session control
- Scalable and modular design
